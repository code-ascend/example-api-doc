# Документация для примера реализации

---

## 1. Группы программ

### Получить все группы программ

```
GET /program-group
```

Возвращает массив групп программ, каждая из которых содержит вложенный массив `programs` — список программ, доступных для бронирования (гарантированно валидных).

### Получить конкретную группу программ

```
GET /program-group/{uuid}
```

**Пример:** `GET /program-group/6323018-f78d-11f0-bbdb-000c298e23d`

Возвращает одну группу программ с вложенным списком программ, доступных для бронирования.

---

## 2. Программы

### Получить все программы

```
GET /program
```

Возвращает полный список программ. Каждая программа содержит поле `idGroup` для связи с соответствующей группой.

### Получить конкретную программу

```
GET /program/{uuid}
```

**Пример:** `GET /program/6c8b3018-f78d-11f0-bbdb-000c298e3a0d`

Возвращает данные конкретной программы, включая поле `idGroup` для связи с группой.

---

## 3. Номера

### Получить все номера

```
GET /room
```

Возвращает массив номеров, доступных для бронирования (гарантированно валидных).

### Получить конкретный номер

```
GET /room/{uuid}
```

**Пример:** `GET /room/2c2bee21-4a69-11f0-bbd8-000c298e3a0d`

Возвращает подробную информацию о конкретном номере.

## 4. Цены и бронирование

### Календарь минимальных цен

```
GET /price/calendar
```

Возвращает минимальные цены на все доступные даты вперёд (глубина зависит от настроек модуля). Скидки не учитываются, конкретная программа и номер не указываются — метод предназначен исключительно для отображения календаря.

**Пример ответа:**

```json
[
  {
    "date": "2026-03-20",
    "minPrice": 6800.00
  },
  {
    "date": "2026-03-21",
    "minPrice": 6800.00
  }
]
```

### Детальные цены

```
GET /price/detailed-prices
```

Возвращает все доступные комбинации «номер + программа» за указанный диапазон дат с подробной информацией о ценах и скидках. Расчёт выполняется для одного гостя в виде приблизительного прайса. Эти данные мы будем выводить в модуле в графе "цена от..."

> **Важно:** Данный метод дополнительно заменяет собой проверку доступности номеров (`/kint/hs/KintEM/availableRooms`). В ответ попадают только те комбинации, в которых номер реально свободен на запрашиваемые даты. Отдельно проверять доступность номеров не нужно.

**Параметры запроса:**

| Параметр    | Тип    | Обязательный | Описание        |
|-------------|--------|:------------:|-----------------|
| `dateStart` | string | ✅           | Дата заезда     |
| `dateEnd`   | string | ✅           | Дата выезда     |
| `idRoom`    | UUID   | ❌           | UUID номера     |
| `idProgram` | UUID   | ❌           | UUID программы  |

**Логика фильтрации:**

- Указан только `idRoom` — возвращаются все доступные программы для этого номера.
- Указан только `idProgram` — возвращаются все доступные номера для этой программы.
- Ни один фильтр не указан — возвращаются все возможные комбинации номеров и программ на выбранные даты.
- Во всех случаях номера, не имеющие свободных мест на указанные даты, **исключаются из ответа**.

**Пример ответа:**

```json
[
  {
    "roomUuid": "0954b7a7-...",
    "programUuid": "885f9bc6-...",
    "minPricePerNight": 7000.00,
    "minPriceTotal": 35000.00,
    "discount": 15
  },
  {
    "roomUuid": "195427a7-...",
    "programUuid": "285f3bc6-...",
    "minPricePerNight": 3000.00,
    "minPriceTotal": 41000.00,
    "discount": 10
  }
]
```

### Финальная цена

```
POST /price/final-price
```

Рассчитывает итоговую стоимость проживания с учётом всех параметров: выбранного номера, дат заезда/выезда и списка гостей (каждый гость может иметь свою программу).

**Тело запроса:**

```json
{
  "personList": [
    {
      "type": "child",
      "age": 3,
      "idProgram": "6c8b3018-f78d-11f0-bbdb-000c298e3a0d"
    },
    {
      "type": "adult",
      "age": 20,
      "idProgram": "6c8b3018-f78d-11f0-bbdb-000c298e3a0d"
    },
    {
      "type": "adult",
      "age": 30,
      "idProgram": "6c8b3018-f78d-11f0-bbdb-000c298e3a0d"
    }
  ],
  "dateStart": "2026-03-28",
  "dateEnd": "2026-03-31",
  "idRoom": "2c2bee21-4a69-11f0-bbd8-000c298e3a0d"
}
```

**Поля `personList`:**

| Поле        | Тип    | Описание                          |
|-------------|--------|-----------------------------------|
| `type`      | string | Тип гостя: `adult` или `child`    |
| `age`       | int    | Возраст гостя                     |
| `idProgram` | UUID   | UUID выбранной программы для гостя|

## 5. Бронирование

### Создать бронирование

```
POST /reservation/create
```

Создаёт бронирование на основе рассчитанной стоимости

```json
{

  "dateEnd": "2026-03-31",
  "idRoom": "2c2bee21-4a69-11f0-bbd8-000c298e3a0d",
  "typePayment": 1,
  "comment": "Просьба поселить на высоком этаже",
  "personList": [
    {
      "firstName": "Иван",
      "lastName": "Иванов",
      "secondName": "Иванович",
      "phone": "79001234567",
      "email": "ivanov@mail.ru",
      "country": "RU",
      "type": "adult",
      "age": 35,
      "idProgram": "6c8b3018-f78d-11f0-bbdb-000c298e3a0d"
    },
    {
      "firstName": "Мария",
      "lastName": "Иванова",
      "secondName": "Петровна",
      "phone": "79009876543",
      "email": "ivanova@mail.ru",
      "country": "RU",
      "type": "adult",
      "age": 30,
      "idProgram": "6c8b3018-f78d-11f0-bbdb-000c298e3a0d"
    },
    {
      "firstName": "Петя",
      "lastName": "Иванов",
      "secondName": "Иванович",
      "phone": "79001234567",
      "email": "ivanov@mail.ru",
      "country": "RU",
      "type": "child",
      "age": 3,
      "idProgram": null
    }
  ]
}
```

**Поля запроса:**

| Поле          | Тип    | Обязательный | Описание                                      |
|---------------|--------|:------------:|-----------------------------------------------|
| `dateStart`   | string | ✅           | Дата заезда (формат `Y-m-d`)                  |
| `dateEnd`     | string | ✅           | Дата выезда (формат `Y-m-d`)                  |
| `idRoom`      | UUID   | ✅           | UUID номера                                   |
| `typePayment` | int    | ✅           | Тип оплаты: `1` — онлайн, `2` — на месте     |
| `comment`     | string | ❌           | Комментарий к бронированию (макс. 250 символов)|
| `personList`  | array  | ✅           | Список гостей                                 |

**Поля `personList`:**

| Поле         | Тип    | Обязательный | Описание                                    |
|--------------|--------|:------------:|---------------------------------------------|
| `firstName`  | string | ✅           | Имя                                         |
| `lastName`   | string | ✅           | Фамилия                                     |
| `secondName` | string | ❌           | Отчество                                    |
| `phone`      | string | ✅           | Телефон (российский формат, 11 цифр)        |
| `email`      | string | ✅           | Email                                       |
| `country`    | string | ✅           | Код страны (ISO 3166-1 alpha-2, например `RU`) |
| `type`       | string | ✅           | Тип гостя: `adult` или `child`              |
| `age`        | int    | ✅           | Возраст гостя                               |
| `idProgram`  | UUID   | ✅*          | UUID программы (*`null` для детей до 4 лет) |

> **Примечание:** При создании бронирования автоматически отправляется уведомление в Telegram и создаётся лид в Bitrix24 CRM на стороне нашего модуля
